# Laboratorio DotNet 

Implementación de Monolito con patron MVC y DAO


## Tech Stack ⚒️
- .NET 8
- MS SQL Server 2022
- REST
- Swagger 3
## Run Locally 🐋

Clone the project

```bash
git clone https://github.com/Estebans441/personapi-dotnet
```

Go to the project directory

```bash
cd personapi-dotnet
```

Run docker compose

```bash
docker-compose up --build -d
```

Go to the url

```bash
http://localhost:5233
```
